Exercise tracker

Create a New User

POST /api/users

Add exercises

POST /api/users/:_id/exercises

GET user's exercise log: GET /api/users/:_id/logs?[from][&to][&limit]

[ ] = optional

from, to = dates (yyyy-mm-dd); limit = number




Enter a username and submit, then copy the given ID, so that you can make entries to the right. from, to, and limit need to be, respectively: a valid date, a valid date, and a positive integer. Example formatting for query values: from=1969-4-20&to=2001-9-11&limit=420. Each value is delimited by &.
